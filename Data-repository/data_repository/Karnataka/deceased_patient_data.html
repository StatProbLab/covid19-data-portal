Name of the CSV File

deceased_patient_data.csv

Basic Description

This csv file contains information on each patient who has passed away due to COVID-19 in Karnataka.

Data Duration

13-03-20 to 31-12-22

List of Column Names with Meanings

Column Name Meaning
Sno Serial number.
District The district that patient belongs to.
State.P.No The State P Code for that patient.
Age.In.Years The age of the patient in years.
Sex The sex of the patient.
Description The description of the patient including why they were tested and which cluster they fall in.
Symptoms The symptoms of the patients.
Co.Morbidities Any co-morbidities that the patients might have had.
DOA The date of admission. The date at which they were admitted to the hospital. For the patients who were brought dead or passed away at their residence, that has been mentioned.
DOD The date of death. The date on which they passed away.
MB.Date The date on which they appeared in media bulletin.
Notes Additional notes if any.
