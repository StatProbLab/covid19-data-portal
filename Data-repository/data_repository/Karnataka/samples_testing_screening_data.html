Name of the CSV File

samples_testing_screening_data.csv

Basic Description

This csv file contains information on the testing and screening done by the Karnataka government. The rows represent dates from 12th March, 2020 onwards.

Data Duration

12-03-20 to 09-02-22

List of Column Names with Meanings

Column Name Meaning
Daily Samples Tested Samples tested on the particular date mentioned in the date column.
Daily samples reported as negative Among the samples that are tested for the day, it show samples that are reported negative.
Daily Samples Positive for COVID-19 Among the samples that are tested for the day, it show samples that are reported positive.
People in Observation The count of people in the observation.
KIA-Passengers The number of passengers screened at the KIA airport.
Mangalore-Passengers The number of passengers screened at the Mangalore airport.
Seaport M and K- passengers The number of patients screened at the seaports in Mangalore and Karwar.
Date The date on which samples are tested.
