Name of the CSV File

Master.csv

Basic Description

This csv file contains information about age-wise and gender-wise travel history of patients, comorbidities, sympttoms etc., across all districts in Karnataka.

Data Duration

13-3-2020 to 23-02-2023

List of Column Names with Meanings

Column Name Meaning
District List of all 31 districts of the Karnataka state.
State.P.No Patient number given state-wise.
Age.In.Years Age of the patient.
Sex Gender of the patient.
Description Travel history associated with the patient.
Symptoms Symptoms observed in the patient.
Co.Morbidities Medical history of the patient.
DOA Date of admission tot he hospital.
DOD Date of death.
MB.Date Date when case was registed to the data repository.
Notes Recovery/Death data of the patient.
