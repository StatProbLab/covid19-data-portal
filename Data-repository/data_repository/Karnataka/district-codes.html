Name of the CSV File

India_Infected.csv

Basic Description

This csv file contains counts of the infected, recovered and deceased patients states wise in India. Each row represents a state. For each date, there are 3 columns representing the total number of infected patients, the total number of recoveries and the total number of deceased patients.

Data Duration

10-03-20 to 01-02-23

List of Column Names with Meanings

Column Name Meaning
District Name List of all 31 districts of the Karnataka state.
TINF The total number of infected patients.
TREC The total number of recovered patients.
TDEC The total number of deceased patients.
