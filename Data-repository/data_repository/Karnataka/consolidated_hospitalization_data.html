Name of the CSV File

consolidated_hospitalization_data.csv

Basic Description

This csv file contains the total counts of active, recovered, discharged, deceased and ICU patients.

Data Duration

09-03-20 to 09-02-22

List of Column Names with Meanings

Column Name Meaning
Date Date starting from 09-03-2020 to 09-02-2022.
Total Active(without ICU, ICUV, ICVO) The number of patients that are hospitalized on the given date, but do not require ICU, ICUV or ICUO.
Total discharged The total number of patients discharged till the corresponding date.
Total deceased The total number of deceased patients till the corresponding date.
Total ICU The count of patients in ICU (intensive care unit), ICUV (ICU and ventilator) or ICUO (ICU and oxygen) on the given date.
